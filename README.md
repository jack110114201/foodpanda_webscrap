# 一、專題摘要
1. 專題主題 : Foodpanda網站爬蟲
2. 基本目標 :
   *  獲得餐廳販售的品項名稱、價格…等資訊
   *  爬取的資訊寫入csv檔，再導入postgres db
3. 進階目標:
   * 串接Kakfa，爬取的資訊寫至producer，再透過consumer將資料導入postgres db
